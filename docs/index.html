实拍视频审核管理系统

本页面用于 GitHub Pages 展示与引导。前端真实页面位于项目的 templates/index.html，需要配合本地后端服务使用。

快速开始

  1. 在本机启动后端服务（建议端口 3001）：
    PORT=3001 python3 start_simple.py
  2. 打开本地页面：http://localhost:3001/

从 GitHub Pages 连接本地后端（可选）

若需在 GitHub Pages 上直接打开前端并连到本地后端，请确保浏览器允许从 HTTPS 页面访问 http://localhost，并且服务已开启 CORS（代码已内置简单 CORS 头）。
