视频审核管理系统

基于Flask + Bootstrap的现代化视频生产审核工作流管理平台

Python Flask Bootstrap 5 SQLite JavaScript

系统概述

这是一个完整的视频生产审核管理系统，将传统的Excel离线审核流程线上化，提升生产审核效率。

核心功能
  • 视频任务管理
  • 多阶段审核流程
  • 实时状态跟踪
  • 批量操作支持
  • 截图标注功能
  • 文件上传下载
用户角色
  • 标注审核员
  • UED审核员
  • 加艺术字人员
  • 项目管理员

审核工作流程

1. 视频生产

原始生产视频上传到平台

2. 标注审核

标注审核员审核原始视频质量

3. 加艺术字

审核通过后添加艺术字效果

4. UED审核

UED审核员审核最终视频效果

5. 审核完成

所有审核通过，视频可发布

6. 审核不通过

返回修改或重新制作

部署说明

本地部署
  1. 克隆项目到本地
  2. 安装Python依赖：pip install -r requirements.txt
  3. 初始化数据库：python init_database.py
  4. 导入Excel数据：python import_excel_data.py
  5. 启动服务：python start_simple.py
  6. 访问：http://localhost:3000
